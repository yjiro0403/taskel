# Taskel AI 実装計画書 (v2)

## ゴールの概要
要件定義書に基づき、「Taskel AI」機能を実装します。
機能実装に加え、本計画では重要な**フェーズ 0: コードベースのスキャンとドキュメント基盤構築**を含めました。これにより、AIと人間が共有されたコンテキストを持ち、将来の開発における効率化を図ります。

## ユーザー確認事項
> [!IMPORTANT]
> この計画には「ワンタイム・フルスキャン（One-Time Full Scan）」戦略が含まれています。
> 現状のプロジェクト構造（たとえ大部分が空であっても）の包括的なスキャンを行い、基盤となるドキュメント（`CLAUDE.md`、`Architecture.md`）を生成します。この初期投資により、以降のタスクで毎回コード全体を読み込ませる必要がなくなり、コストと時間を節約できます。

## 変更内容（Proposed Changes）

### フェーズ 0: コードベースのスキャンとドキュメント基盤構築 (Initial Full Scan)
**ゴール**: 現状のシステムに対する共有理解を確立し、AIのための「メモリ」を作成する。

#### [NEW] ドキュメント成果物
- **`CLAUDE.md` / `AI_CONTEXT.md`**: 
    - AIアシスタントのための包括的なガイド。
    - 内容: プロジェクト概要、技術スタック、コーディング規約、ディレクトリ構造の説明、主要なアーキテクチャ上の決定事項。
    - **メリット**: AIは将来のタスクでリポジトリ全体をスキャンする代わりに、この単一ファイルを読むだけで済みます。
- **`Architecture.md`**:
    - ハイレベルなシステム設計、データフロー図、コンポーネント関係図。
    - 計画中の「Taskel AI」統合アーキテクチャを反映して更新します。
- **`Current_Status_Report.md`**:
    - 現状の実装ステータスのスナップショット
    - 再利用可能な既存コードの特定。

#### [UPDATE] 既存の手順書
- `Docs/`内のセットアップガイドや手順書をレビューし、実際のコードベースの現実に合わせて更新・修正します。

### フェーズ 1: AIサイドバーとコアコンポーネント
**ゴール**: チャットインターフェースと基本的タスク管理UIを実装する。

#### [NEW] UIコンポーネント
- `src/components/ai/AISidebar.tsx`: メインのチャットインターフェース
- `src/components/ai/TaskCreationCard.tsx`: AI提案タスクの「プレビューカード」
- `src/components/tasks/TaskItem.tsx`: タスク表示コンポーネント
- `src/components/tasks/TaskList.tsx`: タスクリスト

#### [NEW] AI ロジック
- `src/app/api/chat/route.ts`: チャットリクエストを処理するAPIルート
- `src/lib/ai/prompts.ts`: AIの振る舞いを定義するシステムプロンプト（時間感覚校正、時間管理OSのペルソナ）
- `src/lib/ai/tools.ts`: ツール定義（createTask, breakdownGoal 等）

### フェーズ 2: 機能統合（反復開発）
**ゴール**: AIと実際のTaskel機能を接続する。

#### ステップ 2.1: タスク作成と時間見積もり
- `TaskCreationCard`を`useStore`に接続。
- 自然言語解析による`estimatedMinutes`（見積もり時間）抽出の実装。

#### ステップ 2.2: 目標分解と紐づけ (Goal Breakdown & Alignment)
- Firestoreから「Goals（目標）」を取得するロジックの実装。
- AIによる目標紐づけ機能の追加。

## 検証計画 (Verification Plan)

### 自動テスト
- **ドキュメント確認**: `CLAUDE.md`と`Architecture.md`が存在し、正確であることを確認。
- **ビルド検証**: `npm run build`を実行。

### 手動検証
1.  **コンテキスト効率性**:
    - 次のタスクで、詳細なコンテキストを与えずにプロジェクトに関する質問をし、AIが`CLAUDE.md`を読んで回答できるか試す。
2.  **プロジェクトセットアップ**:
    - `npm run dev`を実行し、アプリがロードされるか確認。
3.  **AIインタラクション**:
    - AIサイドバーが表示され、機能することを確認。
